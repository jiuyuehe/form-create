# form-create

帮我写一个vue3 +elementplus form组件 的 json schame 生成表单 ，表单可以提取出对应的结构的数据结果，schame+结果可以展示表单数据的功能完整的应用； 我的大体要求如下：

### 第一：整体UI 界面

1、 表单卡片界面，上面是工具栏，主要有创建表单按钮，数据分析按钮。
2、内容区是表单卡片界面， 每设计一个完成一个表单形成一个卡片，卡片带有预览、编辑、删除、数据查看4个按钮。
3、点击数据查看进入到表单scheme 的result 界面，用表格的形式展示结果，有数据编辑（在对应的表单中弹出预览和编辑界面）删除操作，上方有toolbar 主要是搜索操作；
4、其他交给你自由发挥，务必保持界面精美简洁，如果你还不理解，可以参考问卷调查这种应用；


### 第二：重点表单设计界面要求如下： 

#### 1、 通过表格组件+下拉组件的方式填充自定义的方式json schame 的生成。
- 表格为3列， 第一列：用户输入组件的label 名， 第二列，下拉选择 组件主要是 input、 select、object（里面可添加子表单（也是input、select）注意这里的input 就是一个字符串的字段，select就是用户提起设定的值；第三列就是默认值；注意input 有多种;
- 这里我强调一下，你重点的关注的组件是用户输入input（短文本、长文本、数字、手机、邮箱、时间、ip地址等规则）、用户选择 select （选择一个或者多个，设计器中可以提前预设值）、文件上传。
- 这个表格配置完成后生成一个json schame。这个schame 用来反编译成一个表单。
- 总结一下： 点击创建表单界面出来一个表格，可以一行一行的指定 form scheme 的具体实现，可以预览form界面，设计完成之后保存，保存成一个schema json； 

#### 2、表单用来抽取数据，表单数据来源是
这个表单生成的schame 能提供给大模型，作为数据结构去对文本文件内容，合同等作内容的抽取，抽取出来的结果形成个schameresult 对象， 这个result + 表单的schame 能渲染出一个带值的表单，可以编辑，修改，展示；

#### 3、最终呈现给用户的是一个小工具。
第一步先设计表单，第二步上传一个文件或者一段文本，调用ai 抽取文本中的数据，形成对应的schema 的result 集，展示到表格中，每一条数据都能修改。注意这个结果集展示一定是一个表格形式，他需要能支持搜索、排序、导出等功能。

#### 4、 请注意设计的json schema 也能导出，其他用户拿到后导入就是一个完整的表单。
